MediaInfo est un affichage pratique et unifié des informations techniques et les tags les plus pertinents pour les fichiers vidéo et audio.

L'affichage de MediaInfo comprend:
- Conteneur : format, profile, nom commercial du format, durée, débit global, application utilisée, titre, auteur, réalisateur, album, numéro de piste, date...
- Vidéo : format, identifiant de codec, format à l'écran, nombre d'images par seconde, débit, easpace de couleur, sous-échantillonnage chromatique, précision, type de scan, ordre du scan...
- Audio : format, identifiant de codec, fréquence d'échantillonage, nombre de canaux, précision, débit, langue...
- Sous-titres : format, identifiant de codec, langue...
- Chapitres : nombre de chapitres, liste des chapitres...

Les formats supportés par MediaInfo comprennent:
- Conteneur : MPEG-4, QuickTime, Matroska, AVI, MPEG-PS (DVD non chiffrés inclus), MPEG-TS (Blu-ray non chiffrés inclus), MXF, GXF, LXF, WMV, FLV, Real...
- Tags : Id3v1, Id3v2, Vorbis comments, APE tags...
- Vidéo : MPEG-1/2 Video, H.263, MPEG-4 Visual (DivX, XviD inclus), H.264/AVC, Dirac...
- Audio : MPEG Audio (including MP3), AC3, DTS, AAC, Dolby E, AES3, FLAC, Vorbis, PCM...
- Sous-titres : CEA-608, CEA-708, DTVCC, SCTE-20, SCTE-128, ATSC/53, CDP, DVB Subtitle, Teletext, SRT, SSA, ASS, SAMI...

Les fonctionnalités de MediaInfo comprennent:
- Supporte une multitude de format vidéo et audio
- Différents affichages (En arbre, texte)
- Exporte les informations en format texte
- Des versions en interface graphique, ligne de commande, bibliothèque partagée (.dylib) sont disponibles (les version ligne de commande et bibliothèque partagée sont disponibles séparément sur le site de l'éditeur)
- Intégration avec le shell (drag 'n' drop et menu contextuel)
