dvbsnoop - Example Outputs

The following links are displaying sample outputs of dvbsnoop.
Output may vary, due to changes in newer versions.

   


DVBSNOOP "DVB Section Stream" output examples:

  • PAT, Pid 0x00
  • PMT
  • CAT, Pid 0x01
  • NIT, Pid 0x10 (DVB-S)
  • NIT, Pid 0x10 (DVB-C)
  • NIT, Pid 0x10 (DVB-T)
  • SDT, Pid 0x11
  • BAT, Pid 0x11
  • EIT, Pid 0x12
  • TOT, TDT, Pid 0x20
  • user defined tables


DVBSNOOP "DSM-CC / MHP" output examples:

  • AIT
  • Datagram
  • DDB
  • DII
  • DSI
  • INT


DVBSNOOP "MPEG Transport Stream (TS)" output examples:

  • Transport Stream
  • Transport Stream, subdecoding Section
  • Transport Stream, subdecoding PES


DVBSNOOP "MPEG Packetized Elementary Stream (PES)," output examples:

  • Video stream
  • Audio stream
  • Data stream
  • Teletext stream
  • VPS data
  • WSS data


DVBSNOOP "special scan" output examples:

  • PID-SCAN on transponder
  • PID-BANDWIDTH scan
  • Signal strength statistics


DVBSNOOP FrontEnd information:

  • frontend DVB-S
  • frontend DVB-C
  • frontend DVB-T


DVBSNOOP hex / ascii dump and special mode examples:

  • hex / ascii dump modes
  • PID spidering (sections)
